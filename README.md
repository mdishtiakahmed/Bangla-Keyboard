# Bangla Phonetic IME

## How to Build & Install (কিভাবে APK তৈরি করবেন)

Since this is an AI-generated project, you need to build the APK yourself. It is very easy and free.
যেহেতু এটি একটি সোর্স কোড, তাই আপনাকে Android Studio ব্যবহার করে APK টি তৈরি করে নিতে হবে।

### Steps (ধাপসমূহ):

1. **Download & Extract**: Download the ZIP file and extract it.
   (ZIP ফাইলটি ডাউনলোড করে আনজিপ করুন।)

2. **Open in Android Studio**:
   - Open Android Studio.
   - Select **File > Open**.
   - Select the extracted folder.
   (Android Studio ওপেন করে আনজিপ করা ফোল্ডারটি সিলেক্ট করুন।)

3. **Build APK**:
   - Wait for "Gradle Sync" to finish (bottom bar).
   - Go to menu: **Build > Build Bundle(s) / APK(s) > Build APK(s)**.
   (মেনু থেকে Build > Build APK সিলেক্ট করুন।)

4. **Install**:
   - Click "locate" when the build finishes.
   - Copy the `app-debug.apk` to your phone and install.
   (তৈরি হওয়া APK টি আপনার ফোনে নিয়ে ইনস্টল করুন।)
